### C++

- C++의 입출력
  - C++의 입출력 라이브러리는 ios_base를 기반 클래스로 한 ios클래스, ios를 기반으로 한 istream과 ostream, 이 둘을 합친 iostream 등이 있다. 
  
  ```cpp
  #include <iostream>

  int main(void) {
      int num;
      std::cin >> num;
      std::cout << num << std::endl;

      return 0;
  }

  // 입력 : 5
  // 출력 : 5
  ```
